#  Traffic Challan Management System

## Overview
This project is a Traffic Challan Management System developed using Java, JDBC, and MySQL. It helps manage traffic violations, generate challans, and track payments efficiently.

## Technologies Used
- Java
- JDBC
- MySQL

## Features
- Issue challan
- Mark challan as paid
- View unpaid challans
- Daily report generation
- Identify repeat offenders

## Database
- Database name: `traffic_db`
- Table: `challans`

## How to Run
1. Import project into Eclipse/IntelliJ
2. Run `schema.sql` in MySQL
3. Update DB credentials in `DBConnection.java`
4. Run `Main.java`

##  Files Included
- `Main.java` → Main logic
- `DBConnection.java` → Database connection
- `schema.sql` → Database structure
